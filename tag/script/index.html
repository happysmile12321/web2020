我经常看到有的网站 script的type类型是application / javascript, 有的是text / javascript.而我喜欢用text / javascript,或者直接不写type属性.其实text / javascript标签是比较老的写法, 标准的写法是application / javascript, 在HTML5标准中,不写type则默认就是ECMAScript.IE7, IE8用application / javascript会引入失败
